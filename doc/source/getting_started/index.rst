Getting Started
===============

This section is the shortest path into MARIO.

If you are new to the package, the recommended order is:

1. install MARIO and the optional dependencies you need;
2. run the quickstart on a small test database;
3. use the next-steps page to jump to tutorials, how-to guides or developer documentation.
