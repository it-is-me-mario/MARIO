How to Work with a Database
===========================

Common workflows
----------------

Aggregate one or more sets
   Use ``Database.aggregate(...)`` with an aggregation workbook generated by
   ``get_aggregation_excel(...)``.

Export and re-import a database
   Use ``to_excel(...)``, ``to_txt(...)`` or ``to_parquet(...)`` and the
   corresponding parser to roundtrip the data.

Add sectors or activities/commodities
   Use the workbook-driven ``get_add_sectors_excel(...)`` and
   ``add_sectors(...)`` flow.

Transform between SUT and IOT structures
   Use ``to_iot(...)`` or ``to_chenery_moses(...)`` when those transforms are
   meaningful for your data.
