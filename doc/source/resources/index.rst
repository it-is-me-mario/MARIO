Resources
=========

This section collects the material that supports the main guides, tutorials and
API reference, but does not fit naturally inside a single workflow page.

Use it when you need background information around the package rather than a
step-by-step procedure.

In particular, this is where you will find:

* publications and citation material;
* glossary and terminology;
* settings-related notes;
* changelog information;
* contributor-oriented pages and project-maintenance notes.
