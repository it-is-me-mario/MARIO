Publications
============

This page collects the main publications currently bundled with the MARIO
documentation.

How to Cite MARIO
-----------------

When MARIO is used as the software framework behind an analysis, the primary
reference to cite is the software paper below.

.. bibliography:: ../publications/mario.bib
   :filter: False
   :style: unsrt

   Tahavori2023

This is the canonical MARIO citation and should be the default reference unless
your journal or project requires a different citation policy.
