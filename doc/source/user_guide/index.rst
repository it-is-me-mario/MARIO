User Guide
==========

This is the main narrative path through MARIO. It is the best place to start
once the package is installed and you want more context than a quickstart, but
something less notebook-heavy than the tutorials.

The User Guide combines two kinds of material:

* task-oriented guides for common operations such as parsing, aggregation,
  export and structural workflows;
* conceptual pages that explain how MARIO represents sets, blocks, layouts and
  database workflows internally.

In other words, this section is meant to explain both how to use MARIO and how
to think about it.
