Supply-chain analyses
=====================

This section groups practical supply-chain workflows built on top of MARIO's
inspection and calculation APIs.

Use ``Calculate trades`` for region-by-region commodity or sector trade maps,
including multi-scenario comparisons. Use ``Calculate linkages`` for backward
and forward linkage indicators.
