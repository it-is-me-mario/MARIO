Glossary
========

MARIO uses a compact variable nomenclature across IOT and SUT workflows.

.. list-table:: MARIO terminology
   :widths: 20 20 60
   :header-rows: 1

   * - Variable
     - Also known as
     - Meaning
   * - ``Z``
     - ``T``
     - Intersectoral transaction flows matrix
   * - ``z``
     - ``A``
     - Intersectoral transaction coefficients matrix
   * - ``Y``
     - ``F``
     - Final demand matrix
   * - ``X``
     - ``x``, ``q``, ``g``
     - Production vector
   * - ``V``
     - ``F``
     - Factor-of-production transaction flows matrix
   * - ``v``
     - ``f``, ``B``, ``S``
     - Factor-of-production coefficient matrix
   * - ``E``
     - ``F``, ``D_pba``, ``terr``
     - Satellite transaction flows matrix
   * - ``U``
     - ``T``
     - Use transaction flow matrix
   * - ``S``
     - ``V``, ``M``, ``T``
     - Supply transaction flow matrix
   * - ``EY``
     - ``S_Y``, ``F_hh``, ``F_y``
     - Satellite final-demand flow matrix
   * - ``VY``
     - --
     - Factor-of-production final-demand flow matrix
   * - ``M``
     - --
     - Economic impact matrix
   * - ``F``
     - ``D_cba``, ``con``
     - Footprint matrix

For the historical terminology page, see :doc:`terminology`.
